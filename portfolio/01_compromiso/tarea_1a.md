# Tarea 1.A: Contexto para la comunicación

El canal principal de comunicación es el correo electrónico y la herramienta básica [Gmail](https://mail.google.com),
que forma parte de la plataforma educativa del centro.

![Gmail](images/Captura%20de%20pantalla%202025-11-19%20a%20las%2023.44.55.png)

Para facilitar la gestión del correo electrónico, sobre todo cuando se tienen varias cuentas, es recomendable utilizar
clientes de correo especializados como [Spark](https://sparkmailapp.com/es):
